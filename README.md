# 🎮 Jurgo VR - Experiencia de Realidad Virtual

Un proyecto de realidad virtual web utilizando **A-Frame** para crear experiencias inmersivas.

## 🚀 Características

- ✅ Terreno 100x100 metros con grid de referencia
- ✅ Espacio VR vacío listo para desarrollo
- ✅ Iluminación natural configurada
- ✅ Controles de navegación (WASD + ratón)
- ✅ Compatible con dispositivos VR
- ✅ Puntos cardinales para orientación

## 🛠️ Tecnologías

- **A-Frame 1.4.2** - Framework WebVR
- **HTML5** - Estructura
- **JavaScript** - Lógica del juego

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/estebanLopezU/jurgovrprueba.git

# Entrar al directorio
cd jurgovrprueba

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm start
```

## 🎮 Controles

| Acción | Control |
|--------|---------|
| Moverse | WASD o Flechas |
| Mirar | Arrastrar ratón |
| Modo VR | Botón VR en esquina inferior |
